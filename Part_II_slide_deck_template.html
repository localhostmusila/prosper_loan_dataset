Part II - (Dataset Exploration: Loan Data from Prosper)¶

by (Simon Musila)¶

Investigation Overview¶

Some of the variables that I focused on are the LoanStatus, Term, BorrowerAPR, BorrowerRate will be of help in determining the APR for a borrower, factors affecting the the interest rate. Also to determine the loan status we will have to combine variables like LoanOriginalAmount, LoanStatus, EmploymentStatus, IsBorrowerHomeowner. Some of the major questions that were identified to help in the visualization included:

  • What factors affect the Borrower's interest rate for this loan?
  • What factors affect the Borrower's Annual Percentage Rate (APR) for the loan?
  • What are the factors affecting status of the loan?
  • If being a home owner affects the borrowing capacity?
  • What factors affect the original loan amount?
  • What affect the Term limit for a Loan?

Dataset Overview¶

Loan Data from Prosper data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.

Started with Data wrangling which involved assessing data, data cleaning, manipulation and data visualization which was the main work done on the dataset. Dataset was minised to 16 variables but on analysis LoanOriginationDate variable was divided into loanOrigination_month and loanOrigination_year.

BorrowerAPR Distribution¶

BorrowerAPR has 3 peaks (multimodal Distribution) This is observed between the regions:

  • Between 0.15 and 0.2
  • At 0.3
  • Between 0.35 and 0.4

Other regions remain to be evenly distributed.

LoanOriginalAmount vs BorrowerAPR¶

  • BorrowerAPR count highest between 1000 and 7000.
  • We have a negative correlation between BorrowerAPR and LoanOriginalAmount. As LoanOriginalAmount increases, BorrowerAPR is decreasing and BorrowerAPR count is decreasing significantly after 16,000.
  • When LoanOriginalAmount between 1000 - 7000 highest count of BorrowerAPR is 6706 at approximately 0.35.

It can be noted that when LoanOriginalAmount is small it has more borrowers and very high BorrowerAPR.

#Plotting Influence of LoanOriginalAmount on BorrowerAPR (Heatmap)
plt.figure(figsize=(20,8))
bins_x = np.arange(1000,35000+5000,3000)
bins_y=np.arange(0.05,0.40+0.06,0.04)
labels = ['{}'.format(x) for x in bins_x]
gt = plt.hist2d(y='BorrowerAPR',x='LoanOriginalAmount',data=loan,cmin=0.5,cmap='viridis_r',bins=[bins_x,bins_y])
count = gt[0]
for i in range(count.shape[0]):
    for j in range(count.shape[1]):
        annot = count[i,j]
        if annot!=np.nan and annot >= 2000:
            plt.text(x=gt[1][i]+50,y=gt[2][j]+0.005,s=int(annot),color='white')
        elif annot !=np.nan and annot > 0:
             plt.text(x=gt[1][i]+50,y=gt[2][j]+0.005,s=int(annot),color='black')
plt.colorbar()
plt.xticks(bins_x,labels)
plt.title('Influence of LoanOriginalAmount on BorrowerAPR (Heatmap)')
plt.xlabel('LoanOriginalAmount')
plt.ylabel('BorrowerAPR');
